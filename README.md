# homebrew-better-than-you

Homebrew tap for [BetterThanYou](https://github.com/NomaDamas/BetterThanYou) — a CLI-first AI portrait battle tool.

## Install

```bash
brew install NomaDamas/better-than-you/better-than-you
```

That's it. Homebrew auto-resolves the tap from the `<org>/<short>/<formula>` form.

## Manual tap (alternative)

```bash
brew tap NomaDamas/better-than-you
brew install better-than-you
```

## Upgrade

```bash
brew upgrade NomaDamas/better-than-you/better-than-you
```

## Source

The formula sources `v<version>.tar.gz` from the main repo's GitHub Releases. The canonical formula lives in [Formula/better-than-you.rb](Formula/better-than-you.rb).
